## Saron Orthopedic Specialty Center Website

A modern, responsive website for **Saron Orthopedic Specialty Center**, showcasing advanced orthopedic trauma care, joint replacement, and comprehensive musculoskeletal services.

![Saron Orthopedic Specialty Center Homepage](./website%20screenshot.png)

## Overview

This project is a marketing and informational site for Saron Orthopedic Specialty Center.  
It highlights core services, success metrics, and key information about the medical team, programs, and facilities, with clear calls to action for location and contact.

## Key Features

- **Hero Section**
  - Bold branding and tagline for Saron Orthopedic Specialty Center
  - Short description of trauma care, joint replacement, and musculoskeletal services
  - Primary call-to-action buttons for **Location** and **Call Now**

- **Service Highlights**
  - Trauma care
  - Joint replacement
  - Spine care
  - PRP treatment
  - Imaging
  - Laboratory

- **Impact Statistics**
  - 73,000+ people treated
  - 15+ years of service
  - 98% success rate

- **Content Sections**
  - About the center
  - Detailed services
  - Programs
  - Gallery of facilities and procedures
  - Articles and educational content
  - Medical team
  - Contact section

- **User Experience**
  - Clean, professional medical-themed UI
  - Responsive layout optimized for desktop and mobile
  - Sticky navigation for quick access to all main sections

## Tech Stack

- **Framework**: Next.js (App Router) with React
- **Language**: TypeScript
- **Styling**: Modern responsive CSS (e.g. Tailwind CSS or custom CSS modules)
- **Package Manager**: npm or yarn

yarn start## Project Structure (High Level)

- **`src/app`**: Route definitions and pages (e.g. `gallery`, `articles`, `team`, `contact`).
- **`src/components`**: Reusable UI components (services sections, medical gallery, team, stats, navigation).
- **`public`**: Static assets such as images used throughout the site.
- **`website screenshot.png`**: Screenshot of the homepage used in this README.

## Customization

- **Branding**: Update logos, colors, and typography in the main layout and styling files.
- **Content**: Edit text and images in the relevant components (`Services`, `Program`, `Gallery`, `Team`, `Contact`) to reflect up‑to‑date information.
- **SEO**: Configure meta tags, titles, and Open Graph data in the layout or page metadata.
